* 將記帳本的寫入功能完成。
* 必須真的寫進資料庫
* 所有欄位必填

  2. 「日期」不得大於今天
  3. 「備註」最多輸入100個字元（備註為 TextArea）
  4. 列表加入顏色變換
  5. 類型的「支出」字樣顯現為紅色
* 類型的「收入」字樣顯現為藍色
